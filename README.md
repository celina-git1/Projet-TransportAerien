# Projet-TransportAerien

#### Description du projet : 
Ce projet vise à prédire les retards de vols au départ de l'aéroport d'Orly en se basant sur des données historiques. Il utilise des techniques de scraping, de traitement de données et de machine learning pour estimer le risque de retard en fonction de plusieurs variables : compagnie aérienne, destination, horaire, jour de la semaine et conditions météorologiques.
